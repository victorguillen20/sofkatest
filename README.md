# AppSofkaTest
### Prueba para el puesto de: Desarrollador Front SSR
> [!NOTE]
> Se aplicaron los principios de clean code y SOLID, además del concepto de 3 patrones de diseño: Componente Presentacional, Componente Reutilizable y Control Personalizado.

Este proyecto está hecho en Angular version 17.3.17.
Se usó también, DevContainer para desarrollar este proyecto y que sea fácil de volver a levantarla.
<img width="1917" height="1128" alt="imagen" src="https://github.com/user-attachments/assets/aab2df81-5db6-4b46-b4c5-25ed8d4d56be" />

La aplicación cumple con las 6 funcionalidades requeridas además, tiene en cuenta rendimiento, pantallas de precarga (Skeletons) y
responsive design.

También muestra mensajes de succes y error de manera visual. A continuación, un vistazo a las funcionalidades en la app:
